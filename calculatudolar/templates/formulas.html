como se pueden tener los datos de todas esas páginas?

   Seguro te preguntarás "como se hizo para tener los mismos datos que en dolartoday(por ejemplo)?".La respuesta es simple: una API ... ¿y qué es una API? aquí te lo explican mejor, pero para resumir es un "programa/archivo" que proporciona datos y se le puede enviar o bien recibir datos como es este el caso.

Una API se ve como esto, esos datos se procesan y se utilizan como se necesiten, y como se procesan? , simple , JavaScript un lenguaje de programación súper util.

¿y en el caso de no contar con una API?

  Pues en este caso se deben optener los datos de otra manera y la respuesta es web scraping que es eso?, con que se come?, el web scraping es una manera de entrar a un sitio web y obtener información de él.
por ejemplo @bolivarcucuta no posee API, bueno no que yo sepa, por ende se debe ingresar a su cuenta en twitter para poder optener la tasa del dolar en frontera

y con qué se hace eso??, simple , python un lenguaje de programación exelente para aprender a programar, aunque también se puede hacer con PHP, Ruby ,ó incluso javascript con nodejs

ACLARATORIA

  Éste sitio web no busca ser un cambio oficial o referencial del valor del dolar y euro paralelos en Venezuela, el único propósito es el de informar dichos cambios y/o tasas

   Los datos proporcionados son gracias a:

  • airtmrates*
  • dolartoday*
  • @MonitorDolarVe*
  • @bolivarcucuta*

NOTA:   *no se utilizan todos los datos de dichas webs, sólo se usan los más relevantes en este caso, se recomienda visitar sus sitios y/o perfiles para obtener toda la informacion detallada
